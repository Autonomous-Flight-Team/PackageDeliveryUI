# Save Locations
**Mac**
`{user}/Library/Application Support/DroneView`

**Windows**
`%APPDATA%/DroneView`

**Linux**
`~/.config/droneview`

# Data
Inside the DroneView folder there are three subfolders
 - Telemetry (Timestamped telemetry logs)
 - Logs (Timestampted log files)
 - Settings (JSON user setting files)
